# Buff Win Tab Management

- go to the app and open
  - 3 different windows
  - 10 different buffers

- play around for a bit, opening and closing different files
 - swap windows
 - close a windows
 - open a window
 - open buffers in Telescope with `Shift + h`

## Shortcuts

### Buffers
- `<leader>bn` : [B]uffer [N]ext
- `<leader>bp` : [B]uffer [P]revious
- `<leader>bl` : [B]uffer [L]ast (swaps to previous buffer)
- `<leader>bc` : [B]uffer [C]lose
- `<leader>bq` : [B]uffer [Q]uit all others
- `<S-h>`      : Open telescope buffers list

### Windows
- `<leader>wv` or `<C-w>v` : Window Vertical split
- `<leader>wh` or `<C-w>s` : Window Horizontal split
- `<leader>wd` or `<C-w>c` : Window Quit and Realign
- `<C-h>` : Move focus to the left window
- `<C-l>` : Move focus to the right window
- `<C-j>` : Move focus to the lower window
- `<C-k>` : Move focus to the upper window

### Terminal
- `<leader>tt` : Toggle Terminal
- `<Esc><Esc>` : Exit terminal mode (back to normal mode)
