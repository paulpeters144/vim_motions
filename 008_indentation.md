# Indentation

To indent lines, while in "INSERT" mode, you can press
- `<C-t>` to indent
- `<C-d>` to decrease the indent

## Shortcuts

### Insert Mode
- `<C-t>` : Indent current line
- `<C-d>` : Unindent current line

### Normal/Visual Mode
- `>`  : Indent (use with motion or in visual mode)
- `<`  : Unindent (use with motion or in visual mode)
- `==` : Auto-indent current line
- `gg=G` : Auto-indent entire file
