## Shortcuts

### Telescope (Search)
- `<leader><leader>` : Find files
- `<leader>sg`      : Search by Grep
- `<leader>sw`      : Search current Word
- `<leader>s.`      : Search Recent Files
- `<leader>sn`      : Search Neovim files (config)
- `<leader>sd`      : Search Diagnostics
- `<leader>/`       : Fuzzily search in current buffer

### File Explorer
- `-` : Open parent directory (Oil)

### Search & Jump
- `<leader>/` : **Fuzzy search current document** (Best for searching everything at once)
- `/`          : Search forward for a word
- `?`          : Search backward for a word
- `n`      : Jump to next match
- `N`      : Jump to previous match
- `*`      : Search for the word under the cursor (forward)
- `#`      : Search for the word under the cursor (backward)
- `<Esc>`  : Clear search highlights (no hlsearch)
- `<C-o>`  : Jump back to previous location
- `<C-i>`  : Jump forward to next location (after <C-o>)

### Scrolling
- `<C-d>` : Scroll down (centered)
- `<C-u>` : Scroll up (centered)
- `<C-f>` : Scroll page down (centered)
- `<C-b>` : Scroll page up (centered)
