# Read Code Def

For this exercise, go to anywhere in the "admin-app" repo, and look at the definitions of functions and interfaces.

- `gd` - for going to definition
- `Shift + k` - for seeing the definition popup

## Shortcuts (LSP)
- `<leader>cd` : [C]ode [D]efinitions
- `<leader>cR` : [C]ode [R]eferences
- `<leader>ci` : [C]ode [I]mplementations
- `<leader>ct` : [C]ode [T]ype Definition
- `<leader>co` : [C]ode [O]pen Document Symbols
- `<leader>cr` : [C]ode [R]ename
- `<leader>ca` : [C]ode [A]ction
