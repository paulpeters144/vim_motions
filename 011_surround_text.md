# Surround Text

User the below lines practicing putting quotes and `{}` around blocks of text or code.

## Shortcuts (mini.surround)
- `sa` : Add surrounding (e.g., `saiw'` - surround inner word with single quotes)
- `sd` : Delete surrounding (e.g., `sd'` - delete surrounding single quotes)
- `sr` : Replace surrounding (e.g., `sr'"` - replace single quotes with double quotes)
